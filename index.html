Box Generator

Description:

This is a Javascript-based generative web application.
You can use the controls on the right to manipulate the animation by changing the values.

Parameters:

  • Num: this controls the number of rectangles
  • Steps: this controls the size of the rectangles
  • R: this controls the amount of red
  • G: this controls the amount of green
  • B: this controls the amount of blue
  • Alpha: this controls the opacity of the rectangles
  • Roughness: this controls the roughness of the rectangles
  • Text: this control the display text
  • Text size: this controls the size of the text
  • Show description: this toggles the description on.
